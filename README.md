## üöÄ Getting started

Clone the project and access the folder

```bash
$ git clone https://github.com/dyhalmeida/ignite-lab-ds.git && cd ignite-lab-ds
```

Follow the steps below
```bash
# Install the dependencies
$ yarn install

# Start the project
$ yarn dev

# Start the storybook
$ yarn storybook

# Run storybook test
$ yarn test-storybook
```
